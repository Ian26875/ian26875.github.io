|Post Views:

WeatherForecastControllerTests.cs

進行 .net core xUnit 的整合測試時，我們通常會使用 WebApplicationFactory.cs來模擬整個應用程序的啟動和運行。由於測試中經常需要自定義 HTTP Header，因此可以通過實現一個簡單實作DelegatingHandler.cs來記錄請求和回應的詳細信息。也讓執行整合測試的請求與回應可以輸出到 xUnit 中的 Console視窗。
